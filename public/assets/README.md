# Static assets

Store website static assets in this directory (images, logos, icons, downloadable files).

For this Vite app, files placed under `public/assets` are served from `/assets/...` at runtime.

Examples:
- `public/assets/images/hero-truck.jpg` -> `/assets/images/hero-truck.jpg`
- `public/assets/docs/moving-checklist.pdf` -> `/assets/docs/moving-checklist.pdf`
